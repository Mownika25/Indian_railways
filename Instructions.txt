INSTRUCTIONS FOR SPOT TRAIN
----------------------------
1.ENTER VALID TRAIN NUMBER
2.ENTER START DATE OF TRAIN FROM SOURCE(DD-MM-YYYY)
3.PRESS "SPOT TRAIN" BUTTON

INSTRUCTIONS FOR PNR STATUS
----------------------------
1.ENTER VALID PNR NUMBER
2.PRESS "SEARCH" BUTTON 

INSTRUCTIONS FOR FIND TRAINS
----------------------------
1.ENTER THE DATE ON WHICH TRAIN RUNS
2.ENTER VALID SOURCE STATION CODE
3.ENTER VALID DESTINATION STATION CODE
4.PRESS "SEARCH" BUTTON

INSTRUCTIONS FOR AVAILABLE SEATS
--------------------------------
1.ENTER VALID DATE(DD-MM-YYYY).
2.ENTER VALID SOURCE STATION CODE.
3.ENTER VALID DESTINATION STATION CODE.
4.ENTER VALID TRAIN NUMBER FOR WHICH YOU WANT TO CHECK AVAILABILITY.
5.ENTER VALID CLASS CODE(2A->AC2TIER, 3A->AC3TIER, SL->SLEEPER, CC->CHAIR CAR, FC->FIRST CLASS,2S- SECOND SITTING) ENTER CLASS ONLY IF THE TRAIN HAS IT.
6.ENTER TRAVEL QUOTA (GN-GENERAL, PWD-HANDICAPPED...etc).

INSTRUCTIONS FOR TRAIN FARES
-----------------------------

1.ENTER VALID TRAIN NUMBER FOR WHICH YOU WANT TO CHECK FARE.
2.ENTER THE VALID SOURCE STATION CODE FROM WHICH YOU WANT TO CHECK FARE.
3.ENTER THE VALID DESTINATION STATION CODE UPTO WHICH YOU WANT TO CHECK FARE.
4.ENTER VALID CLASS CODE(2A->AC2TIER, 3A->AC3TIER, SL->SLEEPER, CC->CHAIR CAR, FC->FIRST CLASS,2S- SECOND SITTING) ENTER CLASS ONLY IF THE TRAIN HAS IT.
5.ENTER TRAVEL QUOTA (GN-GENERAL, PWD-HANDICAPPED...etc).
6.ENTER SEARCH BUTTON.

INSTRUCTIONS FOR TRAIN ROUTE
-----------------------------
1.ENTER THE VALID TARIN NUMBER.
2.PRESS THE "SEARCH" BUTTON.
